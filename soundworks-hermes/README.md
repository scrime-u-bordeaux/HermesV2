# soundworks-hermes

Dedicated web app for real-time collaborative control of João Svidzinski's _Hermès V2_ interactive piece.  
Based on [soundworks](https://soundworks.dev) and [vue](https://vuejs.org/) frameworks.

## Basic instructions

`cd` into this folder, run `npm install` then :
* `npm run dev` to start a development environment with hot module reloading.
* `npm run build` or `npm run build:production` to create a release build, then `npm run start` to run it
